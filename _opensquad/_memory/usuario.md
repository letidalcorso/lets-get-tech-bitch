# Perfil da Usuária

## Comunicação

- **Nível técnico:** leiga — não tem familiaridade com termos de desenvolvimento
- **Estilo de resposta esperado:** didático, simples, passo a passo com linguagem do dia a dia
- **Evitar:** jargões técnicos sem explicação (branch, push, commit, localhost, etc.)
- **Preferência:** quando der uma instrução, explicar O QUE fazer, não só o comando
- **Exemplo do que NÃO fazer:** "branch gh-pages criado e publicado — ative o GitHub Pages em Settings > Pages > Source"
- **Exemplo do que FAZER:** "Agora precisamos avisar o GitHub que esse site deve ficar visível. É simples, são só 5 cliques. Vou te guiar passo a passo:"
