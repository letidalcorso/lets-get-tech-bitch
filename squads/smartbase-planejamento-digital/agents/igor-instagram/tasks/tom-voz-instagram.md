# Task: Tom de Voz e Linguagem para Instagram

## Objetivo

Criar o guia prático de tom de voz e linguagem para o Instagram da SmartBase, com exemplos concretos de como escrever (e como não escrever) legendas para cada pilar.

## Processo

1. Revisitar o tom de voz do briefing: "sem enrolação", "a sério", premium
2. Adaptar para o contexto informal e dinâmico do Instagram
3. Criar exemplos reais de legendas para cada pilar (✓ e ✗)
4. Definir regras de CTA e hashtags

## Output Format

```markdown
## 3. Guia de Tom de Voz — Instagram SmartBase

### Personalidade em 5 Palavras
[palavra 1] · [palavra 2] · [palavra 3] · [palavra 4] · [palavra 5]

### Princípios de Linguagem

1. **[Princípio 1]:** [explicação + regra concreta]
2. **[Princípio 2]:** [explicação + regra concreta]
3. **[Princípio 3]:** [explicação + regra concreta]
4. **[Princípio 4]:** [explicação + regra concreta]

### Exemplos por Pilar

Para cada pilar criado na Task 1:

---
**[Nome do Pilar]**

✓ Como escrever:
> "[exemplo de legenda boa — completa, com CTA]"

✗ Como NÃO escrever:
> "[exemplo de legenda ruim — e por que está errado]"

---

### Regras de CTA (Call to Action)

O que usar e quando:
- "[CTA 1]" → quando usar
- "[CTA 2]" → quando usar
- "[CTA 3]" → quando usar

O que nunca usar: [lista de CTAs proibidos]

### Estratégia de Hashtags

**Quantidade por post:** X tags
**Tipos de hashtags:**
- [Tipo 1 — nicho]: exemplos
- [Tipo 2 — local]: exemplos
- [Tipo 3 — comunidade Apple]: exemplos

**Hashtags que a marca NUNCA usa:** [lista]
```

## Quality Criteria

- [ ] 5 exemplos de legendas reais (✓) — completas, não apenas "post sobre X"
- [ ] 5 exemplos de como NÃO escrever (✗) — com motivo
- [ ] CTAs concretos e específicos para SmartBase
- [ ] Estratégia de hashtags com exemplos reais
- [ ] Guia aplicável por qualquer pessoa da equipe sem treinamento adicional
